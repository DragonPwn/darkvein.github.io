# Welcome to DarkVein

DarkVein is a next-generation C2 framework designed for sophisticated red team operations. It prioritizes **stealth**, **flexibility**, and **modern usability**.

## Key Features

- **Native API Evasion**: Avoid userland hooks with native syscalls.
- **BOF Support**: Execute COFF object files in memory.
- **Modern Dashboard**: A reactive, graph-based interface.
